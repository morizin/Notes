\section{Linear Search Algorithm}
\newcommand{\A}{8, 6, 12, 5, 9, 7, 4, 3, 16, 18}

Suppose we have an array:

\expandafter\drawArray\expandafter{\A}{3}

\begin{algorithm}
    \caption{Linear Search Algorithm}
    \label{linear_search}
    \begin{algorithmic}[1]
        \Procedure{LinearSearch}{$A, n, x$}
        \For {$i \gets 0$ to $n-1$}
        \If {$A[i] = x$}
        \State Return $i$
        \EndIf
        \EndFor
        \State Return $-1$
        \EndProcedure
    \end{algorithmic}
\end{algorithm}

\expandafter\drawArray\expandafter{\A}{0}

Best Case $\Longrightarrow$ Searching key element is present at index 0.  Then it will take $O(1)$. \\
Worst Case $\Longrightarrow$ Either the element is absent or its at last index. Then it will take $O(n)$. \\
